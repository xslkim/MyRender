# output2 — CPU软渲染 复刻Unity URP（直觉故事版）

> 三个备选产出之一。与 output1 / output3 主题相同（同一个软渲染器、同样三集划分），但这一版的风格、结构、视觉侧重完全独立。

## 这一版的定位

**故事化、零门槛的"直觉优先"讲法。** 不堆代码、不堆公式，而是把渲染讲成一段**旅行**：跟着一个三角形、一束光，走完整条管线。适合完全没接触过图形学、甚至非程序员的观众也能听懂。

- **视觉优先级**：仍遵循"截图优先"——在每个关键结论处都用**程序真实截图/Unity 对照图**落地；但因为讲的是直觉和比喻，过程中大量使用 **AI 概念图**营造画面（相机=眼睛、三角形=旅人、光=画笔）。图解动画只在"必须有流程/时序"时出现。
- **语气**：讲故事、口语化、有画面感。
- **数学/代码**：尽量不出现；只在一两个非懂不可的点上轻轻带过。

## 三集划分（叙事主线）

| 集 | 副标题 | 主线 |
|----|--------|------|
| EP1 总览 | 一台没有显卡的相机 | 渲染是什么、为什么自己写、整条路的全景 |
| EP2 管线 | 一个三角形的旅程 | 跟着一个三角形：变换 → 裁剪 → 铺成像素 |
| EP3 光照 | 一束光的故事 | 跟着一束光：底色 → 法线 → 明暗高光 → 对照 Unity |

## 视觉素材

- AI 概念图：每集若干张，提示词已写在各 `script.md` 的 `--- visual ---` 里（`@visual: image`）。
- 真实截图（`assets/`）：`final_front` `depth` `normal_mapped` `albedo` `wire` `unity_ref_*` `myrender_*` `orbit.mp4`，作为每段旅程的"落地证据"。
